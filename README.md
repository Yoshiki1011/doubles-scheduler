# 🏓 ダブルス組み合わせ表

テニス・ピックルボールのダブルス組み合わせを自動生成するスマホ対応Webアプリです。

## 🌐 デモ

**👉 [https://yoshiki1011.github.io/doubles-scheduler/](https://yoshiki1011.github.io/doubles-scheduler/)**

## ✨ 主な機能

- **自動スケジュール生成** — コート数・人数を入力するだけで10試合分の組み合わせを生成
- **休憩の均等分散** — 全員が均等に休めるよう自動調整（連続休みを防止）
- **ペア・対戦相手の分散** — なるべく同じペア・同じ相手にならないよう200回リトライで最適化
- **試合の追加** — +5試合ずつ追加可能（既存の履歴を引き継いで最適化）
- **選手名入力** — 番号をタップして名前を入力（ひらがな・カタカナのみ）
- **音声アナウンス** — 各試合の▶ボタンで「第1試合。コート1、いちばんとにばん 対 さんばんとよんばん」と読み上げ
- **画像保存** — スケジュール表をCanvasに描画、長押しで保存
- **スマホ対応** — レスポンシブデザイン、横スクロール時もアナウンスボタンは右端に固定

## 📋 対応フォーマット

| 項目 | 内容 |
|------|------|
| コート数 | 1〜10面 |
| 参加人数 | 4〜30人 |
| 初期試合数 | 10試合（追加可能） |

## 🚀 使い方

1. **コート数・参加人数を入力**
2. **「🎯 10試合分を生成」をタップ**
3. 選手番号をタップして名前を入力（任意）
4. 各試合の **▶ボタン** でアナウンス
5. **「📷 画像として保存する」** でスケジュール表を画像化 → 長押しで保存

## 💻 ローカルで動かす

サーバー不要。HTMLファイルをブラウザで開くだけで動作します。

```bash
git clone https://github.com/<your-username>/doubles-scheduler.git
cd doubles-scheduler
open index.html  # macOS
```

## 🛠 技術スタック

- **フロントエンド**: HTML / CSS / Vanilla JavaScript（フレームワーク・外部ライブラリ不使用）
- **音声**: Web Speech API（`speechSynthesis`）
- **画像生成**: Canvas API
- **ホスティング**: GitHub Pages

## 📊 アルゴリズム

- **休みスケジュール**: 休み回数が少ない人を優先 + 連続休みにペナルティ（貪欲法）
- **ペア割り当て**: バックトラッキング + ペア回数スコアリング
- **コート割り当て**: 全ペアリング評価（コート≤3）またはランダムサンプリング（コート>3）で対戦相手の重複を最小化
- **最適化**: 200回リトライし、ペア重複・対戦相手重複が最小の解を採用
