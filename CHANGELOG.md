# Changelog

## [1.0.0] - 2026-05-07

### 初回リリース

#### 機能
- コート数（1〜10）・参加人数（4〜30）を入力して10試合分の組み合わせを自動生成
- 休憩の均等分散（連続休みを防ぐペナルティ付き）
- 同じペアを避ける最適化（バックトラッキング + 200回リトライ）
- 同じ対戦相手を避ける最適化（スコアリングによるコート割り当て）
- 試合の追加（+5試合ずつ、既存履歴を引き継いで最適化）
- 選手名入力（ひらがな・カタカナのみ、タップでモーダル入力）
- 音声アナウンス（ボタン押下で読み上げ、番号は「〇番」、名前は「〇〇さん」）
- 画像保存（Canvasに描画、長押しで保存）
- アナウンスボタンを右端に固定表示（横スクロール時も常に見える）
